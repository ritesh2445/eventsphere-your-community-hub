# EventSphere - Your Community Hub

EventSphere is a modern, responsive platform designed to help people discover, create, and manage community events. Built with React, Tailwind CSS, and Supabase, it provides a seamless experience for orchestrating events with precision.

## Features

- **Event Discovery**: Easily find events happening in your community.
- **Event Management**: Create and manage your own events with a rich dashboard.
- **Real-time Tools**: Stay updated with real-time notifications and event updates.
- **Seamless Auth**: Fast and secure authentication powered by Supabase.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ritesh2445/eventsphere-your-community-hub.git
   cd eventsphere-your-community-hub
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your environment variables:
   Create a `.env` file in the root directory and add your Supabase credentials.

4. Start the development server:
   ```bash
   npm run dev
   ```

## Development Commands

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm run lint`: Lints the source code.
- `npm test`: Runs the test suite.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **Backend**: Supabase
- **UI Components**: Shadcn UI
- **Icons**: Lucide React
- **Testing**: Playwright, Vitest
